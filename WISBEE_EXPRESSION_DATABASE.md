## 使用ガイドライン

### 表現選択の基準
1. **文脈に応じた適切性**
2. **ユーザーの感情状態への配慮**
3. **会話の流れとの整合性**
4. **知的レベルの維持**

### 避けるべき表現
1. 過度に幼稚な表現
2. 連続する同じ表現
3. 文脈に合わない過剰な感情表現
4. 専門的な場面での過度なカジュアル表現

### 推奨される表現
1. 状況に応じた多様なパターン
2. 知的で洗練された言い回し
3. ユーザーとの良好な関係性を築く表現
4. 創造性と独創性を感じさせる表現
