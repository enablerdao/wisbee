# プレスリリース

**2025年6月10日**  
**EnablerDAO**

## ChatGPTより高速・安全・無料 - 完全ローカルAIチャット「Wisbee」正式版リリース

### 月額料金なし、データ漏洩リスクゼロ、オフラインでも動作する次世代AIアシスタント

**東京** - オープンソース開発を推進するEnablerDAOは本日、完全ローカルで動作するAIチャットアプリケーション「Wisbee（ウィズビー）」の正式版をリリースしました。ChatGPTなどのクラウドベースAIサービスとは異なり、すべての処理をユーザーのデバイス上で実行することで、プライバシーを完全に保護しながら高速な応答を実現します。

### 主な特徴

**1. 完全無料・永久利用可能**
- 月額料金や利用制限は一切なし
- ChatGPT Plus（月額約3,000円）相当の機能を無料で提供
- 一度ダウンロードすれば追加費用なしで永続的に利用可能

**2. 究極のプライバシー保護**
- すべてのデータ処理をローカルで実行
- 会話履歴や個人情報が外部サーバーに送信されることは一切なし
- 企業の機密情報も安心して扱える

**3. 高速レスポンス**
- クラウドサービスと比較して2〜5倍の応答速度
- ネットワーク遅延がないため、瞬時に応答開始
- 低スペックマシンでも軽量モデルで快適動作

**4. オフライン対応**
- インターネット接続不要で動作
- 飛行機内、山間部、災害時でも利用可能
- 初回セットアップ後は完全オフライン環境で機能

### 対応プラットフォーム

現在、以下のプラットフォームに対応：
- macOS（Intel / Apple Silicon）
- Windows 10以降
- Linux（Ubuntu 20.04以降）
- iOS版は2025年夏リリース予定

### 技術仕様

Wisbeeは最新のオープンソースLLM（大規模言語モデル）を活用：
- Llama 3.2、Qwen 2.5、Gemma 2など複数のモデルに対応
- ユーザーのハードウェアスペックに応じて最適なモデルを選択可能
- 4GB RAMの低スペックPCから高性能GPUまで幅広く対応
