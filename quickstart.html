🐝 Wisbee クイックスタートガイド

Wisbeeをお使いいただきありがとうございます！このガイドでは、5分以内にWisbeeを起動する方法を説明します。

📥 ステップ1: インストーラーをダウンロード

お使いのOSに合わせてインストーラーをダウンロードしてください：

🍎 macOS用 🪟 Windows用 🐧 Linux用

※ インストーラーサイズ: 約10KB

🚀 ステップ2: インストーラーを実行

macOS / Linux:

python3 wisbee-installer-mac.py

Windows:

python wisbee-installer-win.py

インストーラーは以下を自動的に行います：

  • 必要なディレクトリの作成
  • jan-nano 4Bモデル（2.27GB）のダウンロード
  • llama.cppのセットアップ
  • 起動スクリプトの作成
💡 ヒント: 初回のモデルダウンロードには5-10分かかります。コーヒーでも飲みながらお待ちください！

🎯 ステップ3: Wisbeeを起動

インストール完了後、以下のコマンドでWisbeeを起動できます：

python3 ~/.wisbee/wisbee.py

または、macOSの場合はアプリケーションフォルダからWisbeeを起動できます。

✨ 使い方のヒント

  • プライバシー優先: すべての処理はローカルで行われます
  • オフライン動作: インターネット接続は不要です
  • 学習機能: あなたのフィードバックから学習します
  • 高速推論: M1 Macで30-50トークン/秒

🆘 トラブルシューティング

Python3がインストールされていない場合:

  • macOS: brew install python3
  • Ubuntu/Debian: sudo apt install python3
  • Windows: python.orgからダウンロード
